# PostgreSQL

## MySQL과의 차이점

| 측면     | MySQL                   | PostgreSQL                    |
|----------|-------------------------|--------------------------------|
| 철학     | 빠르고 간단             | 표준 준수, 고급 기능            |
| 대상     | 웹 애플리케이션          | 엔터프라이즈, 분석              |
| 강점     | 단순 읽기 성능           | 복잡한 쿼리 최적화              |
