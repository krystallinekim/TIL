# 변수 (Variable)

- 컴퓨터 메모리(RAM)에 값을 저장하기 위해 할당받는 공간을 변수라고 한다.
- 프로그램에 의해서 할당받은 공간의 값이 수시로 변경될 수 있기 때문에 변수라고 부른다.

## 변수 기초

### 변수의 선언

- 변수를 사용하기 위해서는 먼저 변수를 선언해야 한다.
- 변수의 선언은 어떤 타입의 데이터를 저장할 것인지 그리고 변수의 이름이 무엇인지 기술해야 한다.
- 변수를 선언하는 구문은 `자료형v변수명;`이다.
    
    ```java
    // 정수형 변수 선언
    byte bnum;
    short snum;
    int inum;
    long lnum;
    
    // 실수형 변수 선언
    float fnum;
    double dnum;
    
    // 논리형 변수 선언
    boolean isTrue;
    
    // 문자형 변수 선언
    char ch;
    
    // 문자열 변수 선언
    String str;
    ```
    

### 변수의 명명 규칙

- 영어 대소문자가 구분되며 문자 수의 제한이 없다.
- 숫자로 시작하면 안 된다.
- 특수문자는 '_'와 '$'만 허용한다.
- 여러 단어로 조합된 변수의 이름은 첫 시작 글자는 소문자로 하고 조합되는 단어의 첫 글자를 대문자로 하는 것이 관례이다.
- 의미 있는 이름을 지어주는 것이 좋다.
- 예약어를 사용하면 안 된다. (class, int, new, null 등)

### 변수의 값 대입과 초기화

- 변수에 값을 저장할 때는 대입 연산자(=)를 사용한다.
    
    ```java
    int age;
    
    // 한 개의 변수는 한 개의 데이터만 저장할 수 있다.
    // 연속으로 값을 저장하면 마지막에 대입한 값이 저장된다.
    age = 10;
    age = 20;
    ```
    
- 변수를 선언하고 처음 저장되는 값을 초기값, 변수에 초기값을 대입하는 것을 변수의 초기화라 한다.
- 지역변수(메소드 안에 선언된 변수)를 사용하려면 반드시 초기화되어야 한다. - 지역변수는 변수에 값을 저장해야 메모리에 변수가 올라감
    
    ```java
    // 선언 후 초기화
    int age;
    
    age = 100;
    
    // 선언과 동시에 초기화
    int age = 100;
    ```
    

### 변수의 사용

- 변수는 출력문이나 연산식 내부에서 변수에 저장된 값을 출력하거나 연산할 때 사용한다.
    
    ```java
    int age = 100;
    
    System.out.println("나이 : " + age);
    ```
    

### final 변수

- final 변수는 단 한 번만 값의 대입이 가능하며, 이후에는 다른 값으로 변경할 수 없다.
- `final` 키워드를 사용해서 변수로 선언하면 초기화 이후에 값을 재할당할 수 없다.
- 보통 대문자로 변수명을 표시
    
    ```java
    // final 변수 선언
    final int PI;
    
    PI = 3.14;

    // 선언과 동시에 final 변수 초기화
    final int PI = 3.14;
    ```
- 참조 이후에 다시 재정의하려 하면 에러남

## 자료형 (Data Type)

- 자료형은 데이터의 타입을 의미하며, 변수에 저장할 값의 형태에 맞게 지정해야 한다.

### 데이터 저장 단위

- 비트(bit)
    - 컴퓨터가 데이터를 표현하는 가장 기본적인 단위이다.
    - 0과 1의 2진수 값 하나를 저장할 수 있는 메모리 공간을 의미한다.
- 바이트(byte)
    - 데이터 처리 또는 문자의 최소 단위이다.
    - 8개의 비트가 모여 하나의 바이트가 된다.
    
- 이러한 데이터 저장 단위를 바탕으로, 자바는 다양한 자료형을 제공한다.

### 기본 타입

- 정수, 실수, 문자, 논리 타입을 기본 타입이라고 한다.
- 기본 타입은 정해진 메모리 크기(byte)를 사용해 값을 저장하는데 바이트 크기가 클수록 표현하는 값의 범위가 크다.
    
    | 타입 | 자료형 | 범위 | 크기(byte) | 기본값 |
    | --- | --- | --- | --- | --- |
    | 논리형 | boolean | true, false | 1 | false |
    | 문자형 | char | 0 ~ 65,535 (유니코드) | 2 | '\u0000' |
    | 정수형 | byte | -128 ~ 127 | 1 | 0 |
    | 정수형 | short | -32,768 ~ 32767 | 2 | 0 |
    | 정수형 | int | -2,147,483,648 ~ 2,147,483,647 | 4 | 0 |
    | 정수형 | long | -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807 | 8 | 0L |
    | 실수형 | float | ±1.4E-45 ~ 3.4E38 | 4 | 0.0f |
    | 실수형 | double | ±4.9E-324 ~ 1.8E308 | 8 | 0.0 또는 0.0d |

- long 타입 숫자 뒤에는 대문자 L을 붙여 줘야 함
- float도 double과 구분하기 위해 F를 붙임
- 문자형(char)는 작은따옴표 사용함. 큰따옴표는 문자열

### 참조 타입

- 참조 타입은 값 자체가 아닌 **객체가 저장된 메모리 주소**를 참조하는 타입이다.
- 변수에는 실제 데이터가 아닌 객체의 참조값이 저장된다.
    ```java
    Variable variable = new Variable();  // variable
    ```
- 대충 클래스가 자료형으로 오는 변수는 다 참조타입 변수
- 클래스, 배열, 인터페이스 등이 참조 타입에 해당한다.
- 아무것도 참조하지 않는다는 의미로 null참조가 가능함.
    ```java
    String str = null;
    ```
    

#### 문자열(String)

- 자바에서 큰 따옴표(”)로 감싼 단어나 문장을 문자열(String)이라고 한다.
- 문자열은 클래스 타입으로 문자열 타입의 변수는 참조 변수이다.
- 문자열은 참조 타입이지만(new 써서 만들어도 됨), 기본 타입처럼 사용할 수 있도록 특별히!! 지원된다.
- 기본타입이 아닌 이유는 실행될 때까지 크기 지정이 안되어 있기 때문 -> 메모리를 **동적 할당**함
    
    ```java
    String str = "오리";
    String str = new String("오리");
    String str = "오리" + "꽥꽥"; // 문자열 연결 연산
    String str = new String("오리" + "꽥꽥");
    ```
    

### 리터럴

- 소스 코드에서 프로그래머에 의해 직접 입력된 값을 리터럴이라고 한다.
- 리터럴은 정수, 실수, 문자, 논리, 문자열 리터럴로 구분되며 리터럴들은 정해진 표기법(long: 1000L / 문자: '' / 문자열: "" / ...)대로 작성되어야 한다.
- 리터럴은 변수에 저장될 초기 값을 표현하는 데 사용된다.
    
    ```java
    short s = 32767;
    int i = 100;
    long l = 10000L;
    float f = 0.123f;
    double d = 3.14;
    char c = 'A';
    String str = "ABC";
    boolean b = true;
    ```
    

### 형 변환(Type Casting)

- 데이터의 타입을 변환하는 것을 형 변환(Type Casting)이라 한다. (boolean 제외)
- 자료형 간 표현 범위의 차이에 따라 자동 형 변환, 강제 형 변환으로 나뉜다.

    - 자동 형 변환은 **컴파일러가** 자동으로 값의 범위가 작은 자료형(int)의 값을 범위가 큰 자료형(double)으로 변환한다.

    - 강제 형 변환은 값의 표현 범위가 큰 자료형(double)을 작은 자료형(int)으로 강제로 변환한다.
    
    ```java
        // 자동 형 변환 (단, byte와 short 자료형 값의 계산 결과는 int로 처리한다.)
        double result = 12 + 3.3; // 12.0 + 3.3 = 15.3 (데이터 손실 없음)
    
        // 강제 형 변환
        int result = 12 + (int) 3.3; // 12 + 3 = 15 (0.3 데이터 손실 발생)
    ```
- 나중엔 클래스 타입에서도 형변환이 일어남

### 변수와 메모리 구조

- 자바 프로그램이 실행되면, JVM은 OS로부터 이 프로그램이 필요로 하는 메모리를 할당받고, JVM은 이 메모리를 용도에 따라 여러 영역으로 나누어 관리한다.
- Stack 영역은 메소드가 호출하면 자동 생성되고 메소드가 끝나면 자동 소멸되는 영역으로 지역변수, 매개변수 등이 생성된다.
- Heap 영역은 new 연산자에 의해 동적으로 할당되고 저장되는 공간으로 객체, 배열 등이 생성된다.
- Static(Method) 영역은 클래스와 메소드, static 예약어로 선언된 필드가 저장되는 공간이다.
